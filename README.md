#  Sebajatra - Social Development Events Platform

Sebajatra is a **community-driven platform** where users can create, join, and manage social service events in their local area. The platform encourages social responsibility and community engagement through interactive event management.

##  Features
- View all **upcoming community events**  
- **Create and manage your own events**  
- **Join events** and participate in community activities  
- **Responsive and mobile-friendly design**  
- **Light/Dark theme toggle** with smooth animation using Framer Motion  
- Search and filter events based on type and name  
- Protected routes for **joined events**, **create event**, and **manage events**  


##  Tools & Technologies Used

## Tool / Library ##             ##Purpose##                                             
1. React + Vite                  >>> Fast frontend setup and development               
2. React Router DOM              >>> Routing between pages                              
3. Tailwind CSS / DaisyUI        >>> Styling & responsive design                        |
4. SweetAlert2 / React Hot Toast >> Show success & error messages                      
5. React Icons / Lucide-react    >> Icons for navbar and footer                         
6. Framer Motion                 >> Animations for smooth transitions                  |
7. React Datepicker              >>> Event date selection and validation               

## Pages
**Home / Upcoming Events** – Browse all upcoming events in a grid layout  
**Create Event** – Private page for creating events with title, description, type, date, image, and location  
**Manage Events** – Private page to edit or delete events created by the user  
**Joined Events** – Private page to view events the user has joined  
**Event Details** – View event details and join event functionality
